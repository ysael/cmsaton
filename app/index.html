CMSATHON

In a world where Google documents is for now king!
It is time for a new champion to rise and help humanity in the quest for a better UX.

Rules

You will have 3 weeks starting from today to fullfill those main tasks

  • Create a simple CMS interface
  • Create the front-end template
  • Setup of the preview server

Requirements

For the CMS interface:
  • Secured login
  • Administration pages for the aldo homepage ca_fr and ca_en
  • A strict validation
  • An Image uploader
  • An API in order to retrieve the content in JSON format
For the front-end template:
  • A dynamic live version for development and QA
  • The ability to build a static version
For the preview server:
  • A preview site on our Rackspace server connected in real time to the CMS API
Bonus points:
  • Have a releases/versioning management
  • Advance localisation
Downlaod the JSON examples for the homepage ca_fr and ca_en
